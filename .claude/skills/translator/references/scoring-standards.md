# 翻译质量评分标准

> 通用六维度评分体系，适用于任何领域的英中翻译质量评估。

## 评分维度（总分10分）

### 1. 中文表达自然度 (1.5分)
**核心要求**：译文应符合中文表达习惯，避免英文句式直译

**评分细则**：
- **1.5分（优秀）**：完全符合中文表达习惯，读起来像原创中文文章
  - 示例："Modi gathered legislators" → "莫迪召集议员"（而非"莫迪聚集了议员"）

- **1.0分（良好）**：基本符合中文习惯，偶有英文句式痕迹

- **0.5分（及格）**：能读懂但有明显英文句式，略显生硬

- **0分（不及格）**：严重英文句式，中文读者难以理解

**关键检查点**：
- [ ] 是否使用"对……来说"（应改为"就……而言/对……而言"）
- [ ] 是否使用"有……的"定语从句（应改为中文前置定语）
- [ ] 是否使用"被……"被动语态（优先使用主动表达）
- [ ] 是否使用"在……方面"（多数情况可删除或改写）

---

### 2. 语言客观流畅度 (2.0分)
**核心要求**：译文应客观准确，文字流畅，符合学术翻译规范

**评分细则**：
- **2.0分（优秀）**：语言精准客观，无冗余，节奏流畅

- **1.5分（良好）**：基本客观流畅，偶有冗余或不够精炼

- **1.0分（及格）**：能传达意思但不够精炼，有赘词

- **0分（不及格）**：语言混乱，逻辑不清

**关键检查点**：
- [ ] 是否有"进行""实施""开展"等万金油动词（应使用精确动词）
- [ ] 是否有"的"字连用（"印度的经济的改革"应为"印度经济改革"）
- [ ] 是否有"了""着"等语气词堆砌
- [ ] 是否有"一些""某种程度"等模糊表达（应精确表达）

---

### 3. 动词位置优化度 (1.5分)
**核心要求**：遵循中文"主-动-宾"结构，动词前置，避免动词远离主语

**评分细则**：
- **1.5分（优秀）**：动词紧随主语，句子结构清晰有力

- **1.0分（良好）**：动词位置基本合理，偶有延后

- **0.5分（及格）**：动词距离主语较远，句子头重脚轻

- **0分（不及格）**：动词位置严重错乱，影响理解

**关键检查点**：
- [ ] 主语与核心动词之间是否插入超过10个字的定语
- [ ] 是否将定语从句拆分为独立短句（推荐）
- [ ] 是否使用"，"将长定语后置

---

### 4. 句式简化度 (1.5分)
**核心要求**：避免过多从句嵌套，将复杂长句拆分为多个短句

**评分细则**：
- **1.5分（优秀）**：长句拆分恰当，每句表达一个核心意思

- **1.0分（良好）**：基本简化，偶有长句但不影响理解

- **0.5分（及格）**：长句较多，有2-3层从句嵌套

- **0分（不及格）**：从句嵌套超过3层，严重影响可读性

**关键检查点**：
- [ ] 单句是否超过40字（建议拆分）
- [ ] 是否有"……的……的……的"连续定语嵌套（应拆分）
- [ ] 是否有"虽然……但是……如果……那么……"多重逻辑关系（应分句）

---

### 5. 认知负荷控制 (2.0分)
**核心要求**：降低读者理解难度，信息组织有序，逻辑清晰

**评分细则**：
- **2.0分（优秀）**：信息层次分明，读者可轻松理解
  - 关键措施：
    - 人名首次出现加注音译全称，后续简称
    - 专有名词加括号注释英文或性质
    - 数字统一格式
    - 时间线清晰

- **1.5分（良好）**：基本清晰，偶有信息密度过大的段落

- **1.0分（及格）**：信息堆砌，读者需反复阅读才能理解

- **0分（不及格）**：逻辑混乱，关键信息缺失或矛盾

**关键检查点**：
- [ ] 人名是否首次全称注音，后续简称
- [ ] 专有名词是否注释
- [ ] 数字是否统一格式（"8 per cent"→"8%"）
- [ ] 是否保留原文关键词的英文标注
- [ ] 段落长度是否适中（建议100-150字/段）

---

### 6. 意思准确度 (2.5分)
**核心要求**：译文完整传达原文含义，无遗漏、无曲解、无添加

**评分细则**：
- **2.5分（优秀）**：完全准确，细节无误

- **2.0分（良好）**：核心意思准确，细节略有偏差

- **1.5分（及格）**：大意正确，但有明显信息遗漏或模糊

- **0-1.0分（不及格）**：曲解原意或重大信息遗漏

**关键检查点**：
- [ ] 数字是否精确（"50 per cent"不能译为"约50%"）
- [ ] 时态是否准确（"has announced"现在完成时 vs "announced"一般过去时）
- [ ] 程度副词是否保留（"significantly"应译出"显著"）
- [ ] 否定表达是否准确（"not yet"≠"not"）
- [ ] 专业术语是否准确

---

## 总分计算

**总分 = 各维度得分之和（满分10分）**

**质量等级**：
- **9.5-10分**：优秀翻译，可直接发布
- **9.0-9.4分**：良好翻译，建议微调后发布
- **8.0-8.9分**：合格翻译，需进一步优化
- **7.0-7.9分**：基本合格，需重点修改
- **<7.0分**：不合格，需重译

---

## 使用建议

1. **首次翻译**：完成初稿后立即评分，识别主要问题
2. **迭代优化**：针对得分<1.5分的维度重点修改
3. **终审标准**：仅当总分≥9.5分时生成最终Word文档
4. **记录改进**：每次迭代记录得分变化，分析优化效果
