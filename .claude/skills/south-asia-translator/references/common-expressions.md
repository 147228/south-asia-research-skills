# 翻译惯用表达对照表

> 翻译时参考本表，将口语化/冗长表达替换为书面化/简练表达

## 一、简化表达

### 国家相关
| 口语化 | 书面化 |
|--------|--------|
| 从中国进口 | 自华进口 |
| 从美国进口 | 自美进口 |
| 从印度进口 | 自印进口 |
| 从俄罗斯进口 | 自俄进口 |
| 向中国出口 | 对华出口 |
| 中国和印度之间 | 中印之间 |
| 美国和中国之间 | 中美之间 |

### 时间相关
| 口语化 | 书面化 |
|--------|--------|

| 在过去的几年里 | 近年来 |

| 计划在 | 拟于 |

### 比较相关
| 口语化 | 书面化 |
|--------|--------|
| 相比 | 较 |

## 二、去冗余

### 删除无必要的词
| 冗余 | 简练 |
|------|------|
| 一个重要的 | 重要 |
| 进行了讨论 | 讨论 |
| 作出了决定 | 决定 |
| 取得了进展 | 进展 |
| 实现了增长 | 增长 |
| 的问题 | （直接删除） |
| 的情况 | （直接删除） |

### 避免的冗余结构
| 冗余 | 简练 |
|------|------|
| 使……得到提升 | 提升…… |
| 使……得到改善 | 改善…… |
| 对……进行了 | ……了 |
| 就……而言的话 | 就……而言 |

## 三、去翻译腔

### 被动语态转主动
| 翻译腔 | 自然表达 |
|--------|----------|
| 被认为是 | 被视为/堪称 |
| 被期望 | 有望 |
| 被报道 | 据报道 |
| 它被用于 | 用于 |

### 前提动词
| 翻译腔 | 自然表达 |
|--------|----------|
| 使产品附加值得到提升 | 提高产品附加值 |
| 使双边关系得到改善 | 改善双边关系 |
| 对经济增长产生影响 | 影响经济增长 |
| 对局势造成冲击 | 冲击局势 |

### 避免动名词做主语
| 翻译腔 | 自然表达 |
|--------|----------|
| 提高效率是重要的 | 效率提升至关重要 |
| 加强合作是必要的 | 有必要加强合作 |
| 解决问题需要时间 | 问题的解决需要时间 |

## 四、标点规范

### 区分使用
| 标点 | 使用场景 |
|------|----------|
| 句号。 | 完整句子结束 |
| 逗号， | 句内停顿，不可滥用 |
| 分号； | 并列分句之间 |
| 冒号： | 引出解释或列举 |
| 破折号—— | 解释说明、转折 |
| 省略号…… | 省略或未尽之意 |

### 避免滥用逗号
| 错误 | 正确 |
|------|------|
| 他说，这很重要，因为…… | 他说这很重要，因为…… |
| 印度，作为南亚大国，…… | 印度作为南亚大国，…… |

## 五、数字表达

### 货币
| 原文 | 译文 |
|------|------|
| $10 billion | 100亿美元（约合人民币700亿元） |
| Rs 5,000 crore | 500亿卢比（约合人民币42亿元） |

### 百分比
| 原文 | 译文 |
|------|------|
| increased by 50% | 增长50% |
| fell to 3.5% | 降至3.5% |

### 日期
| 原文 | 译文 |
|------|------|
| January 15, 2024 | 2024年1月15日 |
| Q3 2024 | 2024年第三季度 |
| FY2024 | 2024财年 |

## 六、常见句式

### 引述来源
- 据《xxx》报道，……
- 根据xxx公示/公告/文件显示，……
- xxx在接受采访时表示，……
- 有分析人士指出，……

### 转折过渡
- 然而，……
- 不过，……
- 但值得注意的是，……
- 与此同时，……

### 总结归纳
- 总体而言，……
- 综上所述，……
- 由此可见，……
- 换言之，……

## 七、敏感表述处理

### 涉华表述
| 原文立场 | 处理方式 |
|----------|----------|
| 中国"入侵" | 译为"进入"，或加引号标注原文用词 |
| 中国"威胁" | 译为"中国因素"，或加引号 |
| 西藏"流亡政府" | 加引号，或译为"达赖集团" |

### 涉边境表述
| 原文 | 处理方式 |
|------|----------|
| Arunachal Pradesh | 可译为"阿鲁纳恰尔邦"，必要时注明"我方称藏南" |
| disputed territory | 译为"争议地区" |
| Chinese aggression | 加引号或改为"中方行动" |

### 编者按提示
如原文有明显反华立场，在编者按中加入提示：
> "尤其值得注意的是，作者在文中对中国持批评立场，读者阅读时需加以甄别。"
